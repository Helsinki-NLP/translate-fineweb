| TLS functionality and related terminology<br/>Applies to: Exchange Server 2013<br/>Microsoft Exchange Server 2013 provides administrative functionality and other enhancements that improve the overall management of Transport Layer Security (TLS). As you work with this functionality, you need to learn about some TLS-related features and functionality. Some terms and concepts apply to more than one TLS-related feature. In this topic, a brief explanation of each feature is provided, which is intended to help you understand some differences and general terminology related to TLS and the Domain Security feature set:<br/>Transport Layer Security: TLS is a standard protocol that's used to provide secure Web communications on the Internet or intranets. It enables clients to authenticate servers or, optionally, servers to authenticate clients. It also provides a secure channel by encrypting communications. TLS is the latest version of the Secure Sockets Layer (SSL) protocol.<br/>Mutual TLS: Mutual TLS authentication differs from TLS as TLS is usually deployed. Typically, when TLS is deployed, it's used only to provide confidentiality in the form of encryption. No authentication occurs between the sender and receiver. Additionally, sometimes when TLS is deployed, only the receiving server is authenticated. This deployment of TLS is typical of the HTTP implementation of TLS. This implementation, where only the receiving server is authenticated, is SSL.<br/>With mutual TLS authentication, each server verifies the identity of the other server by validating a certificate that's provided by that other server. In this scenario, where messages are received from external domains over verified connections in an Exchange 2013 environment, Microsoft Outlook displays a Domain Secured icon.<br/>Domain Security: Domain Security is the set of features, such as certificate management, connector functionality, and Outlook client behavior that enables mutual TLS as a manageable and useful technology. Domain Security isn't supported when outbound email is routed through an Exchange 2013 Client Access server.<br/>Opportunistic TLS: In Exchange 2013, Setup creates a self-signed certificate. By default, TLS is enabled. This enables any sending system to encrypt the inbound SMTP session to Exchange. By default, Exchange 2013 also attempts TLS for all remote connections.<br/>Direct trust: By default, all traffic between Edge Transport servers and Mailbox servers is authenticated and encrypted. Again, the underlying mechanism for authentication and encryption is mutual TLS. Instead of using X.509 validation, Exchange 2013 uses direct trust to authenticate the certificates. Direct trust means that the presence of the certificate in Active Directory or Active Directory Lightweight Directory Services (AD LDS) validates the certificate. Active Directory is considered a trusted storage mechanism. When direct trust is used, it doesn't matter if the certificate is self-signed or signed by a certification authority. When you subscribe an Edge Transport server to the Exchange organization, the Edge Subscription publishes the Edge Transport server certificate in Active Directory for the Mailbox servers to validate. The Microsoft Exchange EdgeSync service updates AD LDS with the set of Mailbox server certificates for the Edge Transport server to validate. | TLS-toiminnallisuus ja siihen liittyvä terminologia<br/>Koskee seuraavia: Exchange Server 2013<br/>Microsoft Exchange Server 2013 tarjoaa hallinnollisia toimintoja ja muita parannuksia, jotka parantavat Transport Layer Securityn (TLS) yleistä hallintaa. Kun käytät tätä toimintoa, sinun on opittava joitakin TLS:ään liittyviä ominaisuuksia ja toimintoja. Jotkin termit ja käsitteet koskevat useampaa kuin yhtä TLS:ään liittyvää ominaisuutta. Tässä aiheessa annetaan lyhyt selitys jokaisesta ominaisuudesta, jonka tarkoituksena on auttaa sinua ymmärtämään joitakin TLS:ään ja Domain Security -ominaisuusjoukkoon liittyviä eroja ja yleistä terminologiaa:<br/>Transport Layer Security: TLS on standardiprotokolla, jota käytetään turvallisen verkkoviestinnän tarjoamiseen Internetissä tai intranetissä. Sen avulla asiakkaat voivat todentaa palvelimia tai valinnaisesti palvelimia asiakkaiden todentamiseksi. Se tarjoaa myös suojatun kanavan salaamalla tietoliikenneyhteydet. TLS on uusin versio Secure Sockets Layer (SSL) -protokollasta.<br/>Keskinäinen TLS: Keskinäinen TLS-todennus poikkeaa TLS:stä, koska TLS on yleensä käytössä. Tyypillisesti TLS:ää käytettäessä sitä käytetään vain luottamuksellisuuden tarjoamiseen salauksen muodossa. Lähettäjän ja vastaanottajan välillä ei tapahdu autentikointia. Lisäksi joskus TLS:ää käytettäessä vain vastaanottava palvelin todennetaan. Tämä TLS:n käyttöönotto on tyypillistä TLS:n HTTP-toteutukselle. Tämä toteutus, jossa vain vastaanottava palvelin todennetaan, on SSL.<br/>Molemminpuolisen TLS-todennuksen avulla kukin palvelin varmentaa toisen palvelimen henkilöllisyyden vahvistamalla kyseisen toisen palvelimen antaman varmenteen. Tässä skenaariossa, jossa viestit vastaanotetaan ulkoisilta toimialueilta varmennettujen yhteyksien yli Exchange 2013 -ympäristössä, Microsoft Outlook näyttää Domain Secured -kuvakkeen.<br/>Domain Security: Domain Security on joukko ominaisuuksia, kuten varmenteiden hallinta, liitintoiminnot ja Outlookin asiakaskäyttäytyminen, joka mahdollistaa keskinäisen TLS:n hallittavana ja hyödyllisenä teknologiana. Domain Securityä ei tueta, kun lähtevä sähköposti reititetään Exchange 2013 Client Access -palvelimen kautta.<br/>Opportunistinen TLS: Exchange 2013:ssa asennusohjelma luo itse allekirjoitetun varmenteen. Oletuksena TLS on käytössä. Tämä mahdollistaa minkä tahansa lähettävän järjestelmän salaamaan saapuvan SMTP-istunnon Exchangeen. Exchange 2013 yrittää oletusarvoisesti myös TLS:ää kaikille etäyhteyksille.<br/>Suora luottamus: Oletuksena kaikki liikenne Edge Transport -palvelimien ja Mailbox-palvelimien välillä on autentikoitu ja salattu. Jälleen autentikoinnin ja salauksen taustalla oleva mekanismi on keskinäinen TLS. Sen sijaan, että käytettäisiin X.509-vahvistusta, Exchange 2013 käyttää suoraa luottamusta varmenteiden todentamiseen. Suora luottamus tarkoittaa, että varmenteen olemassaolo Active Directoryssa tai Active Directory Lightweight Directory Services (AD LDS) -palvelussa vahvistaa varmenteen. Active Directorya pidetään luotettavana tallennusmekanismina. Suoraa luottamusta käytettäessä ei ole väliä, onko varmenne itse allekirjoitettu vai varmenneviranomaisen allekirjoittama. Kun tilaat Edge Transport -palvelimen Exchange-organisaatiolle, Edge Subscription julkaisee Edge Transport -palvelinvarmenteen Active Directoryssa, jotta Mailbox-palvelimet voidaan vahvistaa. Microsoft Exchange EdgeSync -palvelu päivittää AD LDS:n Mailbox-palvelinvarmenteiden avulla Edge Transport -palvelimen validoitavaksi. |
| Sulfurimonas autotrophica Inagaki et al. 2003 is the type species of the genus Sulfurimonas. This genus is of interest because of its significant contribution to the global sulfur cycle as it oxidizes sulfur compounds to sulfate and by its apparent habitation of deep-sea hydrothermal and marine sulfidic environments as potential ecological niche. Here we describe the features of this organism, together with the complete genome sequence and annotation. This is the second complete genome sequence of the genus Sulfurimonas and the 15th genome in the family Helicobacteraceae. The 2,153,198 bp long genome with its 2,165 protein-coding and 55 RNA genes is part of the Genomic Encyclopedia of Bacteria and Archaea project.<br/>Strain OK10T (= DSM 16294 = ATCC BAA-671 = JCM 11897) is the type strain of Sulfurimonas autotrophica , which is the type species of its genus Sulfurimonas [1,2]. Together with S. paralvinellae and S. denitrificans, the latter of which was formerly classified as Thiomicrospira denitrificans . There are currently three validly named species in the genus Sulfurimonas [4,5]. The autotrophic and mixotrophic sulfur-oxidizing bacteria such as the members of the genus Sulfurimonas are believed to contribute significantly to the global sulfur cycle . The genus name derives from the Latin word ‘sulphur’, and the Greek word ‘monas’, meaning a unit, in order to indicate a “sulfur-oxidizing rod” . The species epithet derives from the Greek word ‘auto’, meaning self, and from the Greek adjective ‘trophicos’ meaning nursing, tending or feeding, in order to indicate its autotrophy . S. autotrophica strain OK10T, like S. paralvinellae strain GO25T (= DSM 17229), was isolated from the surface of a deep-sea hydrothermal sediment on the Hatoma Knoll in the Mid-Okinawa Trough hydrothermal field [1,2]. Thus, the members of the genus Sulfurimonas appear to be free living, whereas the other members of the family Helicobacteraceae, the genera Helicobacter and Wolinella, appear to be strictly associated with the human stomach and the bovine rumen, respectively. Here we present a summary classification and a set of features for S. autotrophica OK10T, together with the description of the complete genomic sequencing and annotation.<br/>Classification and features<br/>There exist currently no experimental reports that indicate further cultivated strains of this species. The type strains of S. denitrificans and S. paralvinellae share 93.5% and 96.3% 16S rRNA gene sequence similarity with strain OK10T. Further analysis also revealed that strain OK10T shares high similarity (99.1%) with the uncultured clone sequence PVB-12 (U15104) obtained from a microbial mat near the deep-sea hydrothermal vent in the Loihi Seamont, Hawaii . This further corroborates the distribution of S. autotrophica in hydrothermal vents. The 16S rRNA gene sequence similarities of strain OK10T to metagenomic libraries (env_nt) were 87% or less, indicating the absence of further members of the species in the environments screened so far (status August 2010).<br/>Figure 1 shows the phylogenetic neighborhood of S. autotrophica OK10T in a 16S rRNA based tree. The sequences of the four 16S rRNA gene copies in the genome differ from each other by up to four nucleotides, and differ by up to three nucleotides from the previously published sequence (AB088431).<br/>The cells of strain OK10T are Gram-negative, occasionally slightly curved rods of 1.5–2.5 × 0.5–1.0 µm (Figure 2 and Table 1) . On solid medium, the cells form white colonies . Under optimal conditions, the generation time of S. autotrophica strain OK10T is approximately 1.4 h [1,2]. The reductive tricarboxylic acid (rTCA) cycle for autotrophic CO2 fixation is present in strain OK10T, as shown by PCR amplification of the respective genes . Moreover, the activities of several rTCA key enzymes (ACL, ATP dependent citrate lyase; POR, pyruvate:acceptor oxidoreductase; OGOR, 2-oxoglutarase:accecptor oxidoreductase; ICDH, isocytrate dehydrogenase) have been determined, also in comparison to S. paralvinellae and S. denitrificans . There were no enzyme activities for the phosphoenolpyruvate and ribulose 1,5-bisphosphate (Calvin-Benson) pathways detected in strain OK10T , though the latter is apparently active in S. thermophila . Also, soluble hydrogenase activity was not found in strain OK10T . With respect to sulfur oxidation, enzyme activity for SOR (sulfite oxidoreductase) but not for APSR (adenosine 5′-phosphate sulfate reductase) and TSO (thiosulfate-oxidizing enzymes) were detected . A detailed comparison of these enzyme activities to S. paralvinellae and S. denitrificans is given in Takai et al. . Elemental sulfur, thiosulfate or sulfide is utilized as the sole electron donor for chemolithoautotrophic growth with O2 as electron acceptor. Thereby thiosulfate is oxidized to sulfate . Organic substrates and H2 are not utilized as electron donors and only oxygen is utilized as an electron acceptor . Strain OK10T requires 4% sea salt for growth  and is not able to reduce nitrate .<br/>The major cellular fatty acids found in strain OK10T are C14:0 (8.4%), C16:1 cis (45.2%), C16:0 (37.1%) and C18:1 trans (9.4%) . Further fatty acids were not reported . The only polyamine identified in S. autotrophica is spermidine . Spermidine was also found in another representative of the order Campylobacterales, Sulfuricurvum kujiense. For comparison, Hydrogenimonas thermophila, the type species and genus of the family Hydrogenimonaceae in the order Campylobacterales, contains both spermidine and spermine as the major polyamines . The cellular fatty acid composition of S. autotrophica was compared with that of other autotrophic Epsilonproteobacteria from deep-sea hydrothermal vents: Nautilia profundicola AmHT, Lebetimonas acidiphila Pd55T, Hydrogenimonas thermophila EP1-55-1%T, and Nitratiruptor tergarcus MI55-1T . It was found that S. autotrophica strain OK10T has much higher levels of the fatty acid C16:1cis (45.2%) than do other Epsilonproteobacteria from hydrothermal vents express (3.6%–28.8%) [2,30]. On another hand, the percentage of C18:1 trans was the lowest in S. autotrophica: (9.4%), while other Epsilonproteobacteria contained 20.0%-73.3% . C14:0 (8.4%) was also more abundant in strain OK10T than in other strains .<br/>Genome sequencing and annotation<br/>Genome project history<br/>This organism was selected for sequencing on the basis of its phylogenetic position , and is part of the Genomic Encyclopedia of Bacteria and Archaea project . The genome project is deposited in the Genome OnLine Database  and the complete genome sequence is deposited in GenBank. Sequencing, finishing and annotation were performed by the DOE Joint Genome Institute (JGI). A summary of the project information is shown in Table 2.<br/>Growth conditions and DNA isolation<br/>S. autotrophica strain OK10T, DSM 16294, was grown in DSMZ medium 1011 (MJ medium)  at 24°C. DNA was isolated from 0.5–1 g of cell paste using MasterPure Gram Positive DNA Purification Kit (Epicenter MGP04100) following the standard protocol as recommended by the manufacturer, with modification st/LALM for cell lysis as described in Wu et al. .<br/>Genome sequencing and assembly<br/>The genome was sequenced using a combination of Sanger, 454 and Illumina sequencing platforms. All general aspects of library construction and sequencing can be found at the JGI website. Illumina sequencing data was assembled with VELVET , and the consensus sequences were shredded into 1.5 kb overlapped fake reads and used for the assembly with 454 and Sanger data. Contigs resulting from a 454 Newbler (2.0.00.20-PostRelease-11-05-2008-gcc-3.4.6) assembly were shredded into 2 kb fake reads, which were assembled with Sanger data. The Phred/Phrap/Consed software package was used for sequence assembly and quality assessment. After the shotgun stage, reads were assembled with parallel phrap (High Performance Software, LLC). Possible mis-assemblies were corrected with Dupfinisher or transposon bombing of bridging clones (Epicentre Biotechnologies, Madison, WI). Gaps between contigs were closed by editing in Consed, custom primer walk or PCR amplification (Roche Applied Science, Indianapolis, IN) . A total of 790 additional custom primer reactions were necessary to close gaps and to raise the quality of the finished sequence. Illumina reads were also used to improve the final consensus quality using an in-house developed tool - the Polisher . Together, the combination of the Illumina and 454 sequencing platforms provided 155.4 × coverage of the genome. The error rate of the completed genome sequence is less than 1 in 100,000.<br/>Genes were identified using Prodigal  as part of the Oak Ridge National Laboratory genome annotation pipeline, followed by a round of manual curation using the JGI GenePRIMP pipeline . The predicted CDSs were translated and used to search the National Center for Biotechnology Information (NCBI) nonredundant database, UniProt, TIGRFam, Pfam, PRIAM, KEGG, COG, and InterPro databases. Additional gene prediction analysis and functional annotation was performed within the Integrated Microbial Genomes - Expert Review (IMG-ER) platform .<br/>The genome consists of a 2,153,198 bp long chromosome with a 35.2% GC content (Table 3 and Figure 3). Of the 2,220 genes predicted, 2,165 were protein-coding genes, and 55 RNAs; seven pseudogenes were also identified. The majority of the protein-coding genes (69.1%) were assigned with a putative function while the remaining ones were annotated as hypothetical proteins. Autotrofisten ja mikstotrofisten rikkiä hapettavien bakteerien, kuten Sulfurimonas-suvun jäsenten, uskotaan edistävän merkittävästi maailmanlaajuista rikkikiertoa . Sukunimi juontuu latinankielisestä sanasta "rikki", ja kreikan sanasta "monas", joka tarkoittaa yksikköä, osoittaakseen "rikkiä hapettavan sauvan" . Lajin epiteetti juontuu kreikan sanasta 'auto', joka tarkoittaa itseä, ja kreikan adjektiivista 'trofios', joka tarkoittaa imettämistä, hoitamista tai ruokkimista, osoittaakseen sen autotrofiaa . S. autotrophica kanta OK10T, kuten S. paralvinellae kanta GO25T (- DSM 17229), eristettiin syvänmeren hydrotermisen sedimentin pinnasta Hatoma Knollilla Keski-Okinawan Troughin hydrotermisessä kentässä [1,2]. Näin ollen Sulfurimonas-suvun jäsenet näyttävät elävän vapaasti, kun taas muut Helicobacteraceae-suvun jäsenet, suku Helicobacter ja Wolinella, näyttävät liittyvän tiukasti ihmisen vatsaan ja nautakarjaan. Tässä esitellään S. autotrophica OK10T:n yhteenvetoluokitus ja joukko ominaisuuksia sekä täydellisen genomisekvensoinnin ja annotaation kuvaus.<br/>Luokittelu ja ominaisuudet<br/>Tällä hetkellä ei ole olemassa kokeellisia raportteja, jotka viittaisivat tämän lajin edelleen viljeltyihin kantoihin. S. denitrificansin ja S. paralvinellaen tyyppikantojen osuus on 93,5 % ja 96,3 % 16S rRNA -geenisekvenssin samankaltaisuus kannan OK10T kanssa. Lisäanalyysi paljasti myös, että kannalla OK10T on suuri samankaltaisuus (99,1 %) viljelemättömän kloonisarjan PVB-12 (U15104) kanssa, joka on saatu mikrobimatosta lähellä syvänmeren hydrotermistä aukkoa Loihi Seamontissa, Havaijilla . Tämä vahvistaa edelleen S. autotrophian jakautumista hydrotermisissä tuuletusaukoissa. 16S rRNA geenisekvenssi yhtäläisyyksiä kannan OK10T metagenominen kirjastot (env_nt) oli 87% tai vähemmän, mikä viittaa siihen, ettei muita jäseniä lajin ympäristöissä seulotaan toistaiseksi (tilanne elokuu 2010).<br/>Kuvassa 1 näkyy S. autotrophica OK10T:n fylogeneettinen naapurusto 16S rRNA -pohjaisessa puussa. Genomin neljän 16S rRNA -geenikopion sekvenssit eroavat toisistaan jopa neljällä nukleotidilla, ja eroavat jopa kolmella nukleotidilla aiemmin julkaistusta sekvenssistä (AB088431).<br/>Kannan OK10T solut ovat gramnegatiivisia, satunnaisesti hieman kaarevia sauvoja, joiden koko on 1,5 x 2,5 : 0,5 x 1,0 mikrometriä (kuva 2 ja taulukko 1) . Kiinteällä alustalla solut muodostavat valkoisia pesäkkeitä . Optimaalisissa olosuhteissa S. autotrophica -kannan OK10T generointiaika on noin 1,4 h [1,2]. Autotrofisen CO2:n kiinnityksen reduktiivinen trikarboksyylihapposykli (rTCA) esiintyy kannassa OK10T, mikä näkyy vastaavien geenien PCR-vahvistuksena . Lisäksi useiden rTCA:n keskeisten entsyymien (ACL, ATP-riippuvaisen sitraattilyaasin; POR, pyruvaatti:acceptor oksidoreduktaasi; OGOR, 2-oksoglutaraasi:accptor oksidoreduktaasi; ICDH, isosytraattidehydrogenaasi) aktiivisuus on määritetty, myös S. paralvinellaen ja S. denitrificansin vertailussa. Fosfoenolipyruvaatille ja ribuloosi 1,5-bisfosfaatille (Calvin-Benson) ei havaittu entsyymitoimintaa kanta OK10T , vaikka jälkimmäinen on ilmeisesti aktiivinen S. thermophila . Myöskään liukoista hydrogenaasiaktiivisuutta ei löytynyt kannasta OK10T . Rikkioksidaation osalta havaittiin SOR:n (sulfiittioksidoreduktaasi) entsyymiaktiivisuus, mutta ei APSR:n (adenosiini-5-fosfaattisulfaattireduktaasi) ja TSO:n (tiosulfaattia hapettavat entsyymit) . Yksityiskohtainen vertailu näiden entsyymien aktiivisuudesta S. paralvinellae ja S. denitrificans annetaan Takai et al. . Elementaalinen rikki, tiosulfaatti tai sulfidi hyödynnetään ainoana elektronin luovuttajana kemolitoautotrofiseen kasvuun, jossa O2 on elektronin vastaanottaja. Näin tiosulfaatti hapettuu sulfaatiksi . Orgaanisia substraatteja ja H2:ta ei hyödynnetä elektronien luovuttajina ja vain happea hyödynnetään elektronin vastaanottajana . Kanta OK10T vaatii 4 % merisuolaa kasvuun eikä pysty vähentämään nitraattia .<br/>Tärkeimmät solun rasvahapot, joita löytyy kannasta OK10T, ovat C14:0 (8,4 %), C16:1 cis (45,2 %), C16:0 (37,1 %) ja C18:1 trans (9,4 %) . Muita rasvahappoja ei raportoitu . Ainoa S. autotrophidassa tunnistettu polyamiini on spermidiini . Spermidiiniä löytyi myös toisesta Campylobacterales-lahkon edustajasta, Sulfururicurvum kujiensesta. Vertailun vuoksi, Hydrogenimonas thermophila, tyyppilaji ja suku suvun Hydrogenimonaceae järjestyksessä Campylobacterales, sisältää sekä spermidiiniä että spermiiniä suurimpana polyamiinina . S. autotrophilan solun rasvahappokoostumusta verrattiin muihin syvänmeren hydrotermisistä tuuletusaukoista peräisin oleviin autotrofisiin Epsilonproteobakteereihin: Nautilia profundicola AmHT, Lebetimonas acidiphila Pd55T, Hydrogenimonas thermophila EP1-55-1%T, ja Nitratiruptor tergarcus MI55-1T . Todettiin, että S. autotrophica -kannalla OK10T on paljon korkeampi rasvahappo C16:1cis (45,2 %) kuin muilla Epsilonproteobakteereilla hydrotermisistä tuuletusaukoista (3,6 % – 28,8 %) [2,30]. Toisaalta C18:1 transin osuus oli pienin S. autotrophicassa: (9,4 %), kun taas muissa Epsilonproteobakteereissa oli 20,0-73,3 % . C14:0 (8,4 %) oli myös runsaampi kannassa OK10T kuin muissa kantoissa .<br/>Genomisekvensointi ja annotaatio<br/>Genome-projektin historia<br/>Tämä organismi valittiin sekvensointiin sen fylogeneettisen sijainnin perusteella, ja se on osa Genomic Encyclopedia of Bacteria and Archaea -projektia. Genomiprojekti talletetaan Genome OnLine -tietokantaan ja täydellinen genomisekvenssi talletetaan GenBankiin. Sekvensoinnin, viimeistelyn ja annotoinnin suoritti DOE Joint Genome Institute (JGI). Kaikki kirjaston rakentamisen ja sekvensoinnin yleiset näkökohdat löytyvät JGI:n sivuilta. Illumina sekvensointidata koottiin VELVET:in kanssa, ja konsensussekvenssit silputtiin 1,5 kb:n päällekkäisiksi valelukemiksi ja käytettiin kokoamiseen 454 ja Sangerin datan kanssa. 454 Newbler (2.0.00.20-PostRelease-11-05-2008-gcc-3.4.6) -kokoonpanosta syntyneet contigit silputtiin 2 kb:n valelukuihin, jotka koottiin Sangerin datalla. Phred/Phrap/Consed-ohjelmistopakettia käytettiin sekvenssien kokoamiseen ja laadun arviointiin. Haulikkovaiheen jälkeen luennot koottiin rinnakkaisella phrapilla (High Performance Software, LLC). Mahdolliset väärinkäsitykset korjattiin kaksoisolennolla tai transposon-pommituksella sillanrakentajista klooneista (Epicentre Biotechnologies, Madison, WI). Välit contigien välillä suljettiin muokkaamalla Consed, custom primer walk tai PCR-vahvistus (Roche Applied Science, Indianapolis, IN) . Yhteensä 790 ylimääräistä mukautettua primer-reaktiota tarvittiin aukkojen sulkemiseen ja valmiin sekvenssin laadun parantamiseen. Illumina-lukuja käytettiin myös lopullisen konsensuksen laadun parantamiseen käyttämällä talon sisäistä kehitettyä työkalua - puolalaista . | In diets with caloric restriction, the number of brain neurons related to memory grows, we orient ourselves better and capture more details of the environment.<br/>It is no coincidence that every time our body demands food intake, our mobility and mental agility increase. Over thousands of years of evolution, the biological mechanisms that ensure that we get enough nutrients for our body have been enhanced.<br/>Specifically, food demand increases memory. Why? In essence, because it helps to cope with the lack of nutrients.<br/>Among other things, with hunger our ability to orient ourselves in the environment, called spatial memory, increases. For our ancestors, this helped them to remember the path to reach that fruit-laden plant, or the river from which their potential prey drank.<br/>caloric restriction improves memory<br/>When nutrients are scarce, the first to wake up are the neurons of the hippocampus. It is a fundamental brain structure in our declarative memory, which allows us to say things like: "I remember that yesterday I had coffee and toast for breakfast." And it's also an essential part of our spatial memory so that when it's damaged, as it is in patients with Alzheimer's dementia, people don't remember what they did (declarative memory) and become disoriented even in well-known settings like their home (declarative memory). space).<br/>Well, it has been shown that in diets with caloric restriction, the number of neurons in the hippocampus grows and its functionality increases. This makes it especially "sensitive" to changes in the environment so, in addition to orienting ourselves better, we capture more details of it. And of course, it is easier to survive.<br/>It has recently been shown that intermittent fasting can reverse signs of cognitive decline. To prove it, the researchers worked for 36 months with 99 elderly patients, putting them through a calorie-cutting program. When the program ended, they not only lost weight but also reduced insulin levels and signs of inflammation. In addition, they returned to present a cognitive performance according to their age group, leaving behind all the signs of incipient cognitive deterioration.<br/>Another advantage to keep in mind is that the hippocampus helps interpret internal feelings of hunger and coordinates behavior with the body's energy needs. Therefore, patients with damage to the hippocampus (bilateral hippocampal resection) can devour a meal and, if the same dish is presented next, eat it with the same voracity. At the moment we have evidence that, at least, it speeds up memory. This does not mean that everyone should go hungry.
