| How Operating Systems Manage Fonts<br/><br/>In order for an operating system to serve font information to applications and create images of their typefaces onscreen, the fonts have to be "installed." In other words, they have to be stored in the computer in a place where the operating system knows to look for them. In Windows, this storage place is the Windows/Fonts folder. This is not a typical Windows folder in that it contains special tools in the File menu specifically for installing and de-installing fonts. You can also add fonts to this folder by dragging them from their source or by using the File menu's Copy To Folder command.<br/><br/>The Macintosh OS stores fonts in Fonts folders inside of the multiple Library folders found on a typical Mac. One such Library is inside the System folder and contains fonts the OS needs to operate and display its interface correctly. Another Library is in the root hard disk folder, and fonts stored here are available to all users and all applications. In addition, each user of the Mac can create a unique account with personalized settings, and this creates a Library/Fonts folder whose contents are available only to that user. Font files can be dragged or copied into any of these folders. | Kuidas operatsioonisüsteemid hallata fonte<br/><br/>Selleks, et operatsioonisüsteem saaks pakkuda rakendustele fontide teavet ja luua ekraanil nende fontide pilte, peavad fondid olema "installeeritud". Teisisõnu, need tuleb salvestada arvutisse kohas, kus operatsioonisüsteem teab, kuidas neid otsida. Windowsis on see salvestuskoht Windows / Fonts kaust. See ei ole tüüpiline Windowsi kaust, kuna see sisaldab menüüs Fail spetsiaalseid tööriistu, mis on mõeldud fontide installimiseks ja desinstallimiseks. Fondid saate sellesse kausta lisada ka nende allikast lohistades või menüü Fail käsku Kopeeri kausta.<br/><br/>Macintosh OS salvestab fonte fontide kaustadesse, mis asuvad tüüpilises Macis leiduvates mitmes raamatukogu kaustas.
