| Controlling the Global User Directory<br/><br/>The user directory is where Dolphin stores saves, settings, screenshots, and other data. Where the user directory is located depends on the Dolphin version, operating system, and your configuration. In 3.5-2381, the global user directory system was introduced on Windows. By default, the user directory is now a folder in Documents named Dolphin Emulator, but it's possible to configure Dolphin to use the old local user directory or a custom location.<br/><br/>Disable the Global User Directory for a Specific Build with "portable.txt"<br/><br/>To make a build use a local user directory, create a text file named "portable" next to the executable files of the build.
